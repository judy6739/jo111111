أساسيات CSS

مقدمة: CSS (Cascading Style Sheets) هي لغة تنسيق تستخدم لتحديد مظهر صفحات الويب. من خلال CSS، يمكن للمطورين التحكم في تصميم الصفحة، بما في ذلك الألوان، الخطوط، التخطيط، والمسافات.

1. ما هي CSS؟

CSS تُستخدم لفصل محتوى الموقع عن التصميم. يسمح ذلك للمطورين بتغيير تصميم موقع الويب دون الحاجة لتغيير المحتوى الفعلي.

1.1. كيفية استخدام CSS

يمكن استخدام CSS بعدة طرق:

  • الأنماط المضمنة: تُستخدم داخل عنصر HTML باستخدام خاصية style.
  • الأنماط الداخلية: تُستخدم داخل قسم <style> في رأس HTML.
  • الأنماط الخارجية: تُستخدم في ملفات CSS منفصلة تُربط في صفحة HTML.

مثال على الأنماط المضمنة:

<p style="color: blue;">هذا نص أزرق.</p>

مثال على الأنماط الداخلية:

<head>
<style>
p {
color: green;
}
</style>
</head>

مثال على الأنماط الخارجية:

<head>
<link rel="stylesheet" type="text/css" href="styles.css">
</head>

2. القواعد الأساسية في CSS

2.1. تحديد العناصر

تستخدم CSS تحديدات مختلفة لاستهداف العناصر:

التحديد عن طريق الاسم:
p {
color: red;
}
التحديد عن طريق الفئة:
.intro {
font-size: 20px;
}
التحديد عن طريق المعرف:
#header {
background-color: blue;
}

2.2. خصائص CSS

تتضمن CSS مجموعة من الخصائص التي يمكن تطبيقها على العناصر، مثل:

الألوان:
color: red; /* لون النص */
background-color: yellow; /* لون الخلفية */
الخطوط:
font-family: Arial, sans-serif; /* نوع الخط */
font-size: 16px; /* حجم الخط */
font-weight: bold; /* وزن الخط */
التخطيط:
margin: 10px; /* الهوامش */
padding: 5px; /* الحشوة */
border: 1px solid black; /* الحدود */

3. التخطيط باستخدام CSS

3.1. نموذج الصندوق (Box Model)

يعتبر نموذج الصندوق هو الأساس لفهم تخطيط العناصر في CSS. يتكون كل عنصر من:

  • المحتوى: المحتوى الفعلي (النص، الصور).
  • الحشوة: المساحة بين المحتوى وحدود العنصر.
  • الحدود: الخط المحيط بالعنصر.
  • الهامش: المساحة بين حدود العنصر والعناصر الأخرى.

3.2. استخدام Flexbox و Grid

يمكن استخدام تقنيتي Flexbox و CSS Grid لتسهيل التخطيط:

  • Flexbox: يوفر طريقة مرنة لتوزيع المساحة بين العناصر في حاوية.
  • Grid: يسمح بتقسيم الصفحة إلى شبكة من الصفوف والأعمدة.

4. وسائل الإعلام (Media Queries)

تُستخدم وسائل الإعلام لجعل تصميمات CSS تستجيب لأحجام الشاشات المختلفة. مثال:

@media (max-width: 600px) {
body {
background-color: lightblue;
}
}

5. الخاتمة

CSS هي لغة قوية ومهمة لتصميم الويب. بفهم أساسيات CSS، يمكن للمطورين إنشاء صفحات ويب جذابة ومريحة للمستخدمين.
